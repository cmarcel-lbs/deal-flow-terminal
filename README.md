# Deal Flow Terminal

A Bloomberg-style dashboard for tracking early-stage founder activity across Business/Professional Services, AI and Biotech, and Deep Tech Platforms. Built for first-generation founders and investors who want to see the landscape at a glance — recent rounds, rising founders, organic traction, NYC events, social chatter, and white-space gaps.

**Live:** [deal-flow-terminal](https://www.perplexity.ai/computer/a/deal-flow-terminal-0RWPwBNYSyebLPxPCuOW5Q)

## Local Development

```bash
npm install
npm run dev      # serves at http://localhost:5000
npm run build    # outputs to dist/public
```

Built with Vite + React + TypeScript + Tailwind.
